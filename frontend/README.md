# Hi-Five 4번 시안 기반 HTML/CSS/JS

이 버전은 4번 시안의 구조를 기준으로 실제 웹 소스화한 버전입니다.

## 구성
- index.html
- style.css
- script.js
- assets/
  - hero-highway.png
  - control-dashboard.png
  - korea-network-map.png
  - mission-lineart.png
  - contact-highway.png
  - reference-4th-mockup.png

## 사용법
index.html을 브라우저에서 열면 됩니다.

## 기준 시안 비교 기능
우측 하단 `REF` 버튼을 누르거나 키보드 `R`을 누르면 기준 시안 이미지가 반투명하게 오버레이됩니다.
그 상태에서 CSS의 크기/간격을 조정하면 더 정밀하게 맞출 수 있습니다.
